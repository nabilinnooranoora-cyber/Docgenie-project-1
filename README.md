Doc-Genie Project Overview
Doc-Genie is an intelligent Python documentation generator designed to automatically create structured and professional docstrings for Python functions. It uses Abstract Syntax Tree (AST) analysis and advanced code understanding to analyze Python code, extract important details such as function names, parameters, and logic, and generate accurate documentation.

Maintaining documentation manually can be time-consuming and inconsistent across projects. Doc-Genie simplifies this process by automatically generating docstrings while allowing developers to choose different documentation styles such as Google style and NumPy style.

The system provides a simple and interactive interface built with Gradio, allowing users to easily paste Python code and instantly generate documentation.

Technologies used
Python Python is the main programming language used to build the system and handle code processing and

documentation generation.

Abstract Syntax Tree (AST) The AST module is used to analyze Python code and understand function structure, parameters, and logic for accurate docstring generation.

Gradio Gradio provides an interactive web interface where users can input Python code and instantly generate documentation.

ReportLab ReportLab is used to export the generated documentation as PDF files.

Text Processing & Code Analysis Python libraries are used to process text, format docstrings, and support documentation styles like Google and NumPy
